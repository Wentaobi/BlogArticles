## [Ubuntu Server 18.04 安装图形界面](https://jingyan.baidu.com/article/fb48e8bedb5a9d6e622e14b8.html)
### 远程连接Ubuntu Server 18.04，出现如下界面；
### 输入账号密码，登录服务器；
### 更新软件库
apt-get update
### 升级软件
apt-get upgrade
### 安装ubuntu图形界面
apt-get install ubuntu-desktop
### 安装过程比较长，请耐心等待，并按提示操作；
### 重新出现命令输入行后，

### 重启操作系统
reboot
## 重启后按照图形界面进行设置即可！
